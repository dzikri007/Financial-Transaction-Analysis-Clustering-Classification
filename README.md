# Financial-Transaction-Analysis-Clustering-Classification
Tujuan: Membangun model clustering untuk mengidentifikasi pola perilaku transaksi keuangan dan mendeteksi anomali/penipuan (fraud detection).  Pendekatan: Menggunakan algoritma K-Means untuk pengelompokan (unsupervised) dan kemudian menggunakan label tersebut untuk melatih model klasifikasi.
